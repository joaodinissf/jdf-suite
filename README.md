# JDF Suite

A collection of developer tools by [João Dinis Ferreira](https://github.com/joaodinissf).

## Tools

| Tool | Description | Status | Install |
|------|-------------|--------|---------|
| [jdf-hooks](packages/jdf-hooks) | Git hooks framework with interactive CLI | [![PyPI](https://img.shields.io/pypi/v/jdf-hooks)](https://pypi.org/project/jdf-hooks/) | `uvx jdf-hooks setup` |
| [jdf-tab-huddle](packages/jdf-tab-huddle) | Chrome extension — **Huddle** — for sorting, grouping, and de-duplicating browser tabs | v0.1.0 (pre-release, not yet in Chrome Web Store) | Load unpacked `packages/jdf-tab-huddle/src/` in Chrome |
